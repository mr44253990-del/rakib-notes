# Rakib Notes

A modern Android application for taking notes and setting reminders, built with Kotlin and Jetpack Compose.

## Features
- **Google Sign-in**: Secure login using your Google account.
- **Google Drive Integration**: Backup and restore your notes to the cloud.
- **Splash Screen**: 3-page interactive splash screen with developer details.
- **Notes & Reminders**: Create, edit, and delete notes with optional reminders.
- **Material 3 Design**: Clean and modern UI.
